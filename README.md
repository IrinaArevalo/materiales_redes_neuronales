# Materiales de redes neuronales
Materiales de un curso de 6 créditos de Redes Neuronales

Estos materiales cubren la introducción a Redes Neuronales con cierto formalismo matemático. Potenciamos entender el uso de redes neuronales desde los ejercicios "a mano" para después implementarlos en Python. 

**Temario:**
0. Repaso de Álgebra Lineal
1. Introducción al Aprendizaje Profundo
2. Redes Neuronales Convolucionales
3. Redes Neuronales Recurrentes
4. Redes Generativas y Estado del Arte

**Requisitos:**
* Álgebra Lineal
* Estadística descriptiva
* Conocimientos de Python
* (Deseable pero no obligatorio) Un curso introductorio de Aprendizaje Automático/Machine Learning
